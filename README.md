# Construction Cost Analysis Dashboard

## Project Overview
An interactive Power BI dashboard analyzing construction project costs across materials and labor, tracking budget vs. actual spend over a 4-month period to identify cost overruns and inform procurement decisions.

## Tools & Libraries
- Microsoft Excel (data structuring, pivot tables)
- Power BI Desktop (DAX measures, interactive dashboard)

## Key Insights
- **Total Estimated Cost:** ₹10,80,000 | **Total Actual Cost:** ₹11,80,000+ | **Total Overrun:** ₹1.14L+
- **Material costs vastly exceed Labor costs** — Materials account for the large majority of total project spend
- Both Estimated and Actual costs **declined steadily from January to April**, reflecting reduced material/labor needs as the project progressed toward completion
- Actual costs consistently tracked **above estimated costs** across all four months, indicating a systemic underestimation pattern rather than a one-time overrun

## Business Recommendation
Revise estimation methodology for Material costs specifically, since this category drives the majority of both spend and overrun. Building in a 10-15% contingency buffer on material estimates in early project phases (Jan-Feb, where overrun was highest) could improve budget accuracy for future projects.
